# TodoApp-NODEJS-CLI

## Add Task
`app add --title (-t) "title you want to add" --description (--des) "description you want to add"`


## Delete Task
`app add --delete (-d) "title of task which you want delte"`


## Get Task 
`app add --get (-g) "title of task which you want to get"`


## Get All Tasks List
`app list --list (-l)`
